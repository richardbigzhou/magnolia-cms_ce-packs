Welcome to Magnolia

Congratulations! You have successfully set up your Magnolia Bundle. Magnolia comes with two web applications by default: the author instance and the public instance:

Author instance

The Magnolia authoring instance is used for editing content. Administrators and editors have to login to this instance's Admin Central for configuring Magnolia and for adding or modifying pages.

  • Login to the Admin Central* (open in new window)

Note: the default username/password is superuser/superuser.

Public instance

The Magnolia public instance is accessed by your website visitors. It is not accessed by editors directly but administrators are able to login and configure this instance as well.

  • Go to the published website* (open in new window)
  • Login to the Admin Central* (open in new window)

* Before being able to use either of these, you will have to go through their install process. In that case the built-in Magnolia installation wizard will appear right after clicking the link.

For more information, please see the Magnolia documentation. (open in new window)
